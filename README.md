# Beauty-Disposable-Income-Analytics

## Project Overview
This project analyzes the relationship between beauty product trends and disposable income using data analytics techniques.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Analysis Performed
- EDA
- Data Visualization
- Scatter Plot (Income vs Rating)
- Price Distribution Analysis
- Country Comparison

## Key Insights
- Income has weak correlation with ratings
- Mid-range pricing dominates market
- Beauty categories are diversified
